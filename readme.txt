before running the code, do these 4 steps in your system:

Step 1 : download ollama from https://ollama.com/download

Step 2 : Win + R on your system.
	 type cmd and press enter
	 run the command : ollama pull llama3

Step 3 : open a new terminal.
	 run : ollama serve
	 minimize the terminal.

Step 4 : open a new terminal (in the project folder) and run the following command :
	 pip install -r requirements.txt
